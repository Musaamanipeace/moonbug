# **App Name**: LunarEchoes

## Core Features:

- Full Moon Calculation: Calculates the number of full moons visible since the user's birth date, factoring in the date the shape of the moon will be visable
- Current Moon Data: Displays the current date's moon shape, lunar age in cycles, using real-time data.
- User Data Input: Collects user's birth date to calculate lunar cycles from that reference.
- Real-Time Data Integration: Reads current time (24-hour format) and weather data from external API.
- Display Aggregated Info: Displays current time, moon cycles, season, weather, and environmental/nature news.
- AI News Summarizer Tool: Uses an AI tool to generate summarized environmental and nature news.
- Sticky Note Feature: A basic note taking application that helps track and persist information. It may allow users to persist some amount of short notes, memos, and reminders.

## Style Guidelines:

- Primary color: Deep Indigo (#4B0082), reminiscent of the night sky.
- Background color: Very dark desaturated Indigo (#282A3A), providing contrast to the primary color.
- Accent color: Gold (#FFD700), symbolizing lunar glow and highlighting key information.
- Body and headline font: 'Literata', serif, offers a literary and slightly formal feel suitable for displaying diverse data.
- Code font: 'Source Code Pro' for displaying API data or any code snippets.
- Use moon-related icons with a minimalist design.
- Subtle animations for moon phase transitions and data updates.
